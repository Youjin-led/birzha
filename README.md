# ZnakVsem marketplace MVP

Локальный прототип биржи товарных знаков для PatentVsem.

## Что уже есть

- Каталог на 30 стартовых позиций со страницы `https://patentvsem.ru/tovarniy-znak`.
- Пагинация по 24 карточки на страницу.
- Фильтры по МКТУ, скидке и типу бизнеса.
- Поиск с простой RU/EN транслитерацией.
- Избранное через `localStorage`, без регистрации.
- Корзина-заявка: выбранные знаки собираются в форму лида.
- Форма размещения знака: заявка собирается локально, без автопубликации.
- Локальная админка-заготовка: добавляет знаки в браузерное хранилище.

## Что подключить дальше

- Реальную отправку форм в PatentVsem/Tilda/CRM.
- Реальные изображения или логотипы знаков.
- Описания классов МКТУ, когда пользователь предоставит тексты.
- Серверное хранение каталога и настоящую авторизацию администратора.
